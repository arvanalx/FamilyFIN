## 1. Εκκίνηση της Εφαρμογής

Η εφαρμογή τρέχει τοπικά μέσω Python web server.

### Εκκίνηση server

```bash
cd /Users/alex/Documents/Claude/FamilyFIN
python3 server.py
```

Στη συνέχεια ανοίξτε τον browser στη διεύθυνση:

```
http://localhost:8765
```

### Αποθήκευση δεδομένων

Τα δεδομένα αποθηκεύονται **αυτόματα** σε κάθε αλλαγή:
- **Κύρια αποθήκευση:** αρχείο `data.json` στον φάκελο της εφαρμογής (μέσω του server)
- **Εφεδρική αποθήκευση:** `localStorage` του browser (χρησιμοποιείται αν ο server δεν τρέχει)

> ⚠️ Αν ανοίξετε το `index.html` απευθείας (χωρίς server), τα δεδομένα αποθηκεύονται μόνο στο localStorage και μπορεί να χαθούν αν καθαρίσετε τα δεδομένα του browser.

## 3. Λειτουργία Απορρήτου (Privacy Mode)

Το κουμπί **👁️** στην επάνω δεξιά γωνία ενεργοποιεί τη **Λειτουργία Απορρήτου** — χρήσιμη όταν θέλετε να δείξετε κάτι από την εφαρμογή σε τρίτο πρόσωπο χωρίς να αποκαλύψετε τις οικονομικές αξίες.

### Πώς λειτουργεί

| Κατάσταση | Εικονίδιο | Εμφάνιση κουμπιού |
|-----------|-----------|-------------------|
| **Κανονική** | 👁️ | Διαφανές, με περίγραμμα |
| **Απόρρητο** | 🙈 | Μπλε γεμιστό — υπενθυμίζει ότι τα ποσά είναι κρυφά |

Πατήστε μία φορά για ενεργοποίηση, ξανά για απενεργοποίηση. Η εναλλαγή γίνεται με **ομαλό animation θολώματος** (0.3 δευτερόλεπτα).

### Τι αποκρύπτεται

Σε **όλες τις σελίδες** θολώνουν αυτόματα:

| Ενότητα | Στοιχεία που αποκρύπτονται |
|---------|--------------------------|
| **Γρήγορη ματιά** | Υπόλοιπα λογαριασμών (Σήμερα/Μέλλον), Έσοδα/Έξοδα/Υπόλοιπο μήνα, σύνολα καρτών, γραφήματα |
| **Κινήσεις** | Ποσά γραμμών πίνακα, σύνοψη Έσοδα/Έξοδα/Υπόλοιπο |
| **Έσοδα** | Μηνιαία & ετήσια σύνοψη, ποσά γραμμών |
| **Πιστωτικές Κάρτες** | Τρέχον υπόλοιπο, δόσεις/μήνα, μηνιαίες κινήσεις, ποσά κινήσεων |
| **Δόσεις** | Μηνιαία επιβάρυνση, εναπομείναντα ποσό, μηνιαία δόση κάθε δόσης |
| **Συνδρομές** | Μηνιαίο/ετήσιο κόστος, ποσό κάθε συνδρομής |

> 💡 Τα **ονόματα** λογαριασμών, κατηγορίες, περιγραφές και ημερομηνίες παραμένουν ορατά — αποκρύπτονται **μόνο τα νούμερα σε €**.

> ℹ️ Η κατάσταση απορρήτου **δεν αποθηκεύεται** — επαναφέρεται αυτόματα στο κανονικό όταν ανανεώσετε τη σελίδα.

## 4. Γρήγορη Ματιά (Dashboard)

Η αρχική σελίδα παρέχει μια ολοκληρωμένη εικόνα της οικονομικής κατάστασης.

### 4.1 Υπόλοιπα Λογαριασμών

Πίνακας με δύο στήλες για κάθε λογαριασμό:

- **Σήμερα:** Το τρέχον υπόλοιπο όπως έχει καταχωρηθεί στις Ρυθμίσεις
- **Μέλλον:** Υπολογισμένο υπόλοιπο που συνυπολογίζει **όλες τις καταχωρημένες κινήσεις** (έως το τέλος του επόμενου μήνα)

> 💡 Η ημερομηνία "Μέλλον" υπολογίζεται αυτόματα ως η πιο πρόσφατη ημερομηνία κίνησης, με ανώτατο όριο το τέλος του επόμενου μήνα.

Οι τραπεζικοί λογαριασμοί εμφανίζουν το όνομα της τράπεζας σε παρένθεση με μικρότερα γράμματα (π.χ. *Μισθοδοσίας (Alpha Bank)*).

### 4.2 Γράφημα Υπολοίπων

Ραβδόγραμμα που απεικονίζει γραφικά τα υπόλοιπα όλων των λογαριασμών:
- **Έντονο χρώμα** = τρέχον υπόλοιπο (σήμερα)
- **Ανοιχτό χρώμα** = μελλοντικό υπόλοιπο

Κάθε λογαριασμός χρησιμοποιεί το ίδιο χρωματικό ζεύγος (π.χ. σκούρο μπλε / ανοιχτό μπλε) για εύκολη αναγνώριση.

### 4.3 Στατιστικά Μήνα

Σύνοψη για τον **τρέχοντα μήνα**:
- **Έσοδα:** Σύνολο εσόδων από κινήσεις τύπου "Έσοδο"
- **Έξοδα:** Σύνολο εξόδων από κινήσεις τύπου "Έξοδο"
- **Υπόλοιπο:** Διαφορά Εσόδων − Εξόδων (πράσινο αν θετικό, κόκκινο αν αρνητικό)

### 4.4 Πιστωτικές Κάρτες (σύνοψη)

Για κάθε κάρτα εμφανίζεται:
- Τρέχον συνολικό ποσό κινήσεων
- Μηνιαίο ποσό ενεργών δόσεων
- Σύνολο (κινήσεις + δόσεις)

### 4.5 Έξοδα ανά Κατηγορία

**Donut chart** που δείχνει την κατανομή εξόδων του τρέχοντος μήνα από:
- Κινήσεις λογαριασμών (εξαιρούνται κινήσεις με κατηγορία "Κάρτες")
- Κινήσεις πιστωτικών καρτών
- Μηνιαίες δόσεις (ενεργές)

Κάθε τμήμα δείχνει: κατηγορία, ποσοστό (%) και ποσό σε €. Στο κέντρο εμφανίζεται το **συνολικό ποσό εξόδων**.

### 4.6 Τελευταίες Κινήσεις

Οι 6 πιο πρόσφατες κινήσεις. Πατήστε **"Όλες →"** για να μεταβείτε στη σελίδα Κινήσεων.

## 5. Κινήσεις

Η κεντρική σελίδα καταχώρησης και παρακολούθησης **εξόδων και εσόδων** λογαριασμών.

### 5.1 Προσθήκη νέας κίνησης

Πατήστε **"+ Νέα Κίνηση"** (επάνω δεξιά στη σελίδα Κινήσεων).

Συμπληρώστε τα πεδία:

| Πεδίο | Υποχρεωτικό | Περιγραφή |
|-------|------------|-----------|
| **Τύπος** | ✅ | Έξοδο ή Έσοδο |
| **Ημερομηνία** | ✅ | Ημερομηνία κίνησης |
| **Ποσό (€)** | ✅ | Θετικός αριθμός |
| **Περιγραφή** | ✅ | Σύντομη περιγραφή |
| **Κατηγορία** | — | Από τη λίστα κατηγοριών (εξόδων ή εσόδων ανάλογα με τον τύπο) |
| **Λογαριασμός** | ✅ | Ο λογαριασμός χρέωσης/πίστωσης |
| **Σημειώσεις** | — | Προαιρετικό κείμενο |

> 💡 Αλλάζοντας τον **Τύπο** (Έξοδο/Έσοδο), η λίστα κατηγοριών αλλάζει αυτόματα — εμφανίζει Κατηγορίες Εξόδων ή Εσόδων αντίστοιχα.

### 5.2 Επεξεργασία / Διαγραφή κίνησης

- **✏️** — Ανοίγει τη φόρμα επεξεργασίας
- **🗑️** — Ζητά επιβεβαίωση και διαγράφει την κίνηση

### 5.3 Αντιγραφή εσόδου στον επόμενο μήνα

Στις κινήσεις τύπου **Έσοδο**, εμφανίζεται το εικονίδιο **📋**. Πατώντας το, δημιουργείται αυτόματα αντίγραφο της κίνησης με ημερομηνία **+1 μήνα** από την αρχική.

> Χρήσιμο για επαναλαμβανόμενα έσοδα (μισθός, σύνταξη κ.λπ.).

### 5.4 Φιλτράρισμα & Αναζήτηση

Επάνω από τον πίνακα υπάρχουν 4 φίλτρα:

| Φίλτρο | Λειτουργία |
|--------|-----------|
| **Μήνας** | Εμφανίζει κινήσεις συγκεκριμένου μήνα (προεπιλογή: τρέχων μήνας) |
| **Λογαριασμός** | Φιλτράρει ανά λογαριασμό (μόνο οι λογαριασμοί από τις Ρυθμίσεις) |
| **Κατηγορία** | Φιλτράρει ανά κατηγορία εξόδου/εσόδου |
| **🔍 Αναζήτηση** | Αναζήτηση ελεύθερου κειμένου στην περιγραφή |

Τα φίλτρα **συνδυάζονται** μεταξύ τους. Κάτω από τα φίλτρα εμφανίζεται η γραμμή σύνοψης:

- **Έσοδα** (πράσινο) | **Έξοδα** (κόκκινο) | **Υπόλοιπο**

### 5.5 Ταξινόμηση

Πατήστε οποιαδήποτε επικεφαλίδα στηλών (**Ημερομηνία ↕**, **Περιγραφή ↕**, **Ποσό ↕**) για ταξινόμηση. Δεύτερο πάτημα αντιστρέφει τη σειρά.

## 6. Έσοδα

Εξειδικευμένη προβολή που εμφανίζει **αποκλειστικά τα έσοδα** (χωρίς τα έξοδα).

### 6.1 Σύνοψη Εσόδων

- **Μηνιαία Έσοδα:** Σύνολο εσόδων του επιλεγμένου μήνα
- **Ετήσια (εκτίμηση):** Πολλαπλασιάζει τα μηνιαία έσοδα × 12

### 6.2 Προσθήκη νέου εσόδου

Πατήστε **"+ Νέο Έσοδο"**. Τα πεδία είναι ίδια με τη φόρμα κίνησης (χωρίς επιλογή τύπου — είναι πάντα Έσοδο).

### 6.3 Ενέργειες γραμμής εσόδου

- **✏️** — Επεξεργασία
- **📋** — Αντιγραφή στον επόμενο μήνα
- **🗑️** — Διαγραφή

## 7. Πιστωτικές Κάρτες

Αναλυτική προβολή ανά πιστωτική κάρτα.

### 7.1 Επιλογή κάρτας

Στην κορυφή της σελίδας εμφανίζονται **tabs** για κάθε κάρτα που έχει οριστεί στις Ρυθμίσεις. Πατήστε tab για εναλλαγή κάρτας:
- **VISA** — σκούρο μπλε tab
- **MASTERCARD** — σκούρο κόκκινο tab

### 7.2 Σύνοψη κάρτας

Για την επιλεγμένη κάρτα:
- **Τρέχον Υπόλοιπο:** Άθροισμα κινήσεων + μηνιαίων δόσεων
- **Μηνιαίες Δόσεις:** Σύνολο ενεργών δόσεων (/μήνα)
- **Μηνιαίες Κινήσεις:** Κινήσεις του τρέχοντος μήνα

### 7.3 Δόσεις κάρτας

Εμφανίζονται οι ενεργές δόσεις για την επιλεγμένη κάρτα με:
- Γραμμή προόδου (%)
- Αριθμός δόσεων που πληρώθηκαν / σύνολο
- Εναπομένουσες δόσεις
- Μηνιαίο ποσό

**Προσθήκη δόσης:** Πατήστε **"+ Προσθήκη"** (βλ. Ενότητα 8).

### 7.4 Κινήσεις κάρτας

Λίστα όλων των κινήσεων της κάρτας (ταξινομημένες κατά ημερομηνία, πιο πρόσφατες πρώτα).

**Προσθήκη κίνησης κάρτας:** Πατήστε **"+ Κίνηση"** και συμπληρώστε:
- Ημερομηνία, ποσό, περιγραφή, κατηγορία

> 📌 Κινήσεις που δημιουργήθηκαν αυτόματα από **Συνδρομές** εμφανίζονται με το εικονίδιο 🔄.

**Διαγραφή κίνησης:** Πατήστε **🗑️** — αν η κίνηση ανήκει σε συνδρομή, η διαγραφή της **αποτρέπει** την αυτόματη επανεμφάνισή της.

## 8. Δόσεις

Κεντρική σελίδα παρακολούθησης **όλων των ενεργών δόσεων** από όλες τις κάρτες.

### 8.1 Σύνοψη Δόσεων

- **Μηνιαία Επιβάρυνση:** Άθροισμα μηνιαίων δόσεων όλων των ενεργών δόσεων
- **Εναπομείναντα Ποσό:** Συνολικό ποσό που απομένει να πληρωθεί

### 8.2 Προσθήκη νέας δόσης

Πατήστε **"+ Νέα Δόση"**. Συμπληρώστε:

| Πεδίο | Περιγραφή |
|-------|-----------|
| **Περιγραφή** | Όνομα αγοράς (π.χ. "Laptop KOTSOVOLOS") |
| **Συνολικό Ποσό (€)** | Ολική αξία αγοράς |
| **Μηνιαία Δόση (€)** | Ποσό ανά μήνα |
| **Δόσεις που πληρώθηκαν** | Πόσες έχουν ήδη πληρωθεί (αν ξεκινά από το μέσο) |
| **Συνολικές Δόσεις** | Πλήθος δόσεων |
| **Κάρτα** | Η κάρτα στην οποία ανήκει η δόση |
| **Κατάστημα / Πάροχος** | Προαιρετικό |
| **Κατηγορία Εξόδου** | Εμφανίζεται στο γράφημα εξόδων του Dashboard |
| **Ημερομηνία έναρξης** | Μήνας πρώτης δόσης |

### 8.3 Ενέργειες δόσης

- **+1 Δόση** — Καταχωρεί μια δόση ως πληρωμένη (αυξάνει τον μετρητή). Αν συμπληρωθεί ο αριθμός δόσεων, η δόση **απενεργοποιείται** αυτόματα.
- **✏️** — Επεξεργασία στοιχείων
- **🗑️** — Οριστική διαγραφή

### 8.4 Γραμμή προόδου

Κάθε δόση εμφανίζει οπτική γραμμή προόδου που δείχνει το ποσοστό ολοκλήρωσης (πληρωμένες δόσεις / σύνολο).

## 9. Συνδρομές

Διαχείριση **μηνιαίων και ετήσιων συνδρομών** με αυτόματη δημιουργία κινήσεων.

### 9.1 Σύνοψη κόστους

- **Μηνιαία:** Σύνολο μηνιαίων συνδρομών (ετήσιες ÷ 12)
- **Ετήσια:** Συνολικό ετήσιο κόστος όλων των συνδρομών

### 9.2 Προσθήκη νέας συνδρομής

Πατήστε **"+ Νέα Συνδρομή"** και συμπληρώστε:

| Πεδίο | Περιγραφή |
|-------|-----------|
| **Υπηρεσία** | Όνομα (π.χ. "Netflix") |
| **Εικονίδιο** | Emoji για οπτική αναγνώριση (π.χ. 🎬) |
| **Ποσό (€)** | Κόστος ανά περίοδο χρέωσης |
| **Συχνότητα** | Μηνιαία ή Ετήσια |
| **Κάρτα/Λογαριασμός** | Πού χρεώνεται |
| **Ημέρα χρέωσης** | Ημέρα του μήνα (1–31) |
| **Κατηγορία Εξόδου** | Κατηγορία που μεταφέρεται στις κινήσεις |

### 9.3 Αυτόματη δημιουργία κινήσεων

Κάθε συνδρομή **δημιουργεί αυτόματα** κινήσεις για κάθε μήνα:
- Αν η συνδρομή χρεώνεται σε **κάρτα** → δημιουργείται κίνηση στις Κινήσεις Κάρτας
- Αν χρεώνεται σε **λογαριασμό** → δημιουργείται κίνηση στις Κινήσεις Λογαριασμού

Οι αυτόματες κινήσεις αναγνωρίζονται από το εικονίδιο 🔄. Αν διαγράψετε μια τέτοια κίνηση, **δεν** ξαναεμφανίζεται αυτόματα για τον ίδιο μήνα.

> 💡 Αλλάζοντας την **κατηγορία** μιας συνδρομής, ενημερώνονται αυτόματα **όλες οι υπάρχουσες** αυτόματες κινήσεις της.

### 9.4 Ενέργειες συνδρομής

- **✏️** — Επεξεργασία
- **🗑️** — Διαγραφή συνδρομής **και** όλων των αυτόματων κινήσεών της

## 10. Ρυθμίσεις

Κεντρική διαμόρφωση της εφαρμογής. **Οι αλλαγές εδώ επηρεάζουν όλες τις ενότητες.**

### 10.1 Λογαριασμοί

Διαχείριση τραπεζικών λογαριασμών και μετρητών.

**Προσθήκη:** Πατήστε **"+ Νέος"**.

Κάθε λογαριασμός έχει:
- **Όνομα** — Το εμφανιζόμενο όνομα (π.χ. "Μισθοδοσίας")
- **Τύπος** — `Τραπεζικός` ή `Μετρητά`
- **Τράπεζα** — Εμφανίζεται μόνο για τραπεζικούς λογαριασμούς (π.χ. "Alpha Bank")
- **Υπόλοιπο** — Τρέχον υπόλοιπο σε €

**Αποθήκευση:** Πατήστε **"Αποθήκευση"** για κάθε λογαριασμό ξεχωριστά.

**Διαγραφή:** Πατήστε **"Διαγραφή"** (εμφανίζεται μόνο αν υπάρχουν περισσότεροι από έναν λογαριασμός).

> ⚠️ Αλλάζοντας το υπόλοιπο λογαριασμού, ενημερώνεται αμέσως η Γρήγορη Ματιά.

### 10.2 Κάρτες

Διαχείριση πιστωτικών καρτών.

**Προσθήκη νέας κάρτας:** Πατήστε το κουμπί **"+ Νέα"** στην επικεφαλίδα της ενότητας. Δημιουργείται νέα κάρτα με προεπιλεγμένο όνομα την οποία μπορείτε να επεξεργαστείτε αμέσως.

Κάθε κάρτα έχει:
- **Όνομα** — Εμφανιζόμενο όνομα (π.χ. "VISA Gold")
- **Τράπεζα** — Εκδούσα τράπεζα (π.χ. "Alpha Bank")
- **Συνδεδεμένος Λογαριασμός** — Λογαριασμός από τον οποίο γίνεται η εξόφληση της κάρτας

**Αποθήκευση:** Πατήστε **"Αποθήκευση"** μετά από κάθε αλλαγή.

**Διαγραφή:** Πατήστε **"🗑️ Διαγραφή"**. Αν η κάρτα έχει συνδεδεμένες κινήσεις, δόσεις ή συνδρομές, το σύστημα σας ενημερώνει και τα διαγράφει όλα μαζί μετά από επιβεβαίωση.

Οι κάρτες εμφανίζονται ως **tabs** στη σελίδα Πιστωτικών Καρτών.

### 10.3 Κατηγορίες Εξόδων

Λίστα κατηγοριών που χρησιμοποιούνται στις κινήσεις τύπου **Έξοδο**, στις δόσεις και στις συνδρομές.

**Προσθήκη:** Πληκτρολογήστε όνομα στο πεδίο και πατήστε **"Προσθήκη"** (ή Enter).

**Διαγραφή:** Πατήστε το **✕** δίπλα στην κατηγορία.

> ℹ️ Οι προκαθορισμένες κατηγορίες περιλαμβάνουν: Σούπερ μάρκετ, Εστιατόριο/Καφέ, Μεταφορές, Υγεία, Ρουχισμός, Ηλεκτρονικά, Ψυχαγωγία, Τηλεπικοινωνίες, Ενέργεια/Νερό, Ενοίκιο, Ασφάλειες, Δάνειο, Παιδιά κ.ά.

### 10.4 Κατηγορίες Εσόδων

Λίστα κατηγοριών για κινήσεις τύπου **Έσοδο**.

Λειτουργεί ακριβώς όπως οι Κατηγορίες Εξόδων.

> ℹ️ Προκαθορισμένες: Μισθός, Ελεύθερος Επαγγελματίας, Σύνταξη, Ενοίκιο (είσπραξη), Επίδομα, Άλλο.

## 11. Διαχείριση Δεδομένων

Στην ενότητα **Ρυθμίσεις → Δεδομένα** υπάρχουν τρεις επιλογές:

### 📤 Εξαγωγή JSON

Κατεβάζει αρχείο `familyfin_backup_YYYYMMDD.json` με **όλα** τα δεδομένα της εφαρμογής (λογαριασμοί, κινήσεις, δόσεις, συνδρομές κ.λπ.).

> ✅ Συνιστάται περιοδικά ως αντίγραφο ασφαλείας.

### 📥 Εισαγωγή JSON

Φορτώνει αρχείο backup που εξήχθη προηγουμένως. **Αντικαθιστά πλήρως** τα τρέχοντα δεδομένα.

> ⚠️ Η εισαγωγή είναι **μη αναστρέψιμη**. Κάντε πρώτα Εξαγωγή αν θέλετε να διατηρήσετε τα τρέχοντα δεδομένα.

### 🗑️ Επαναφορά

Διαγράφει **όλα** τα δεδομένα και επαναφέρει τις εργοστασιακές προεπιλογές. Ζητά επιβεβαίωση πριν εκτελεστεί.

## Παράρτημα — Λογική Υπολογισμών

### Μελλοντικό Υπόλοιπο Λογαριασμού

```
Μελλοντικό Υπόλοιπο = Τρέχον Υπόλοιπο
                     + Σύνολο Εσόδων (όλες οι μελλοντικές κινήσεις)
                     − Σύνολο Εξόδων (όλες οι μελλοντικές κινήσεις)
```

Λαμβάνονται υπόψη **όλες** οι καταχωρημένες κινήσεις ανεξαρτήτως ημερομηνίας, με ανώτατο όριο το τέλος του επόμενου μήνα.

### Σύνολο Κάρτας (Dashboard)

```
Σύνολο Κάρτας = Άθροισμα κινήσεων κάρτας
              + Άθροισμα μηνιαίων δόσεων (ενεργές)
```

### Γράφημα Εξόδων ανά Κατηγορία

Συνυπολογίζει για τον **τρέχοντα μήνα**:
1. Κινήσεις τύπου Έξοδο (εκτός κατηγορίας "Κάρτες")
2. Κινήσεις πιστωτικών καρτών
3. Μηνιαίο ποσό ενεργών δόσεων (με την κατηγορία τους)
